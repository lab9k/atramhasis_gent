dGrowl
======

A simple "growl" like notifications widget using the dojo js framework and default css3 transitions for the animation effects.

View the <a href="http://stratease.github.io/dGrowl/">github page</a> for a <a href="http://stratease.github.io/dGrowl/#try">demo</a> and the <a href="http://stratease.github.io/dGrowl/#api">API</a> docs.
